# Indian-Banking-Sector-Financial-Analysis-Dashboard
Developed an interactive dashboard using Microsoft Excel to analyze financial performance of major Indian banks Evaluated key metrics such as revenue growth, profitability, and financial ratios Used pivot tables, charts, and slicers to create dynamic data visualizations
Project Features
Interactive dashboard with slicers for filtering data
Multi-year financial analysis of banks
Visualization using charts and pivot tables
Comparison of key financial indicators
Methodology
The project was developed using the following approach:
Collection of financial data for selected banks
Organization of data into structured tables
Creation of pivot tables for analysis
Development of charts for visualization
Use of slicers for interactive filtering
Key Metrics Analyzed
Revenue Growth
Profitability
Financial Ratios
Year-wise Performance
